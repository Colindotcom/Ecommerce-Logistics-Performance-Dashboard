# Ecommerce-Logistics-Performance-Dashboard

## Project Overview
Analysed 10,999 real shipment records from an e-commerce company to identify delivery performance gaps, warehouse inefficiencies, and customer satisfaction drivers using Microsoft Power BI.

## Key Findings
- 60% of all shipments were delayed - systemic operational failure
- Warehouse A identified as critical outlier with highest delay rate (~43%) and lowest customer rating (2.96/5)
- High-importance products delayed at the same rate as low-importance items - no priority dispatch protocol in place
- Average customer rating of 2.98/5 - below the 3.0 threshold, indicating widespread customer dissatisfaction
  
## Tools Used
Microsoft Power BI (Power Query, DAX Measures, Interactive Dashboard)

## Data Source
Kaggle, E-Commerce Shipping Dataset by Prachi Gopalani (10,999 records)
